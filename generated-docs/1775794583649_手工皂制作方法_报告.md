[2] ### 第七步：包装与保存
脱模后的手工皂应放在阴凉干燥处，避免阳光直射。建议使用保鲜膜包裹，可保存6-12个月。

## 常见问题与解决
- 问题1：皂体表面有气泡 → 入模前多喷酒精，入模后轻轻震动模具。
- 问题2：皂体过硬 → 基础油加少了，下次增加5-10毫升。
- 问题3：皂体过软不成形 → 基础油太多或冷却不足，减少基础油或延长冷却时间。
- 问题4：精油气味不明显 → 精油添加

[3] ## 材料准备
制作手工皂需要准备以下材料：
- 皂基：500克，建议使用透明皂基或白色皂基，根据喜好选择。
- 基础油：50毫升，常用橄榄油、椰子油、甜杏仁油、葡萄籽油等。橄榄油滋润度高，椰子油起泡性好。
- 精油：10-20滴，可选薰衣草、茶树、甜橙、薄荷等，增加香气和功效。
- 色素：适量，液体或粉末状，用于调色。
